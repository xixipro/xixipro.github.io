网页的建立

工具：eclipse JDK tomcat maven

准备环境

1.下载eclipse

2.下载安装jdk，配置系统环境才能使用eclipse

Windows系统：我的电脑--高级配置--环境配置
            配置JAVAHOME、classpath等
使用doc命令 java -version 或 javac 验证

3.下载安装tomcat分为文件、eclipse插件、maven倒入插件形式 配置环境

4.下载安装maven 配置环境

新建项目

Group id 项目的域名
Artifact id 项目名称
Packaging 包的类型

web项目的基本设置

在properties（设置）
1.resource中设置编译
2.若不是maven项目，将所需的包通过Javabuildpath 导入到项目中。
3.Javacompile 设置项目的Java的版本
4.若建的是maven项目，maven项目默认JDK是1.5，可在mven–project facets中调整Java版本
5.server和targeted runtimes 选择tomcat
6.若是web项目，默认是没有web.xml文件需要右键项目找打javaee tools就会消除报错。

web.xml

web.xml是项目的入口。
通过项目的-项目IP地址+端口+项目名或者是域名访问项目
默认的欢迎页面：自动搜索相应的页面，这样一个简单的网页搭建成功。
welcome-file-list
welcome-file
html/index.html
