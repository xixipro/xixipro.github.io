springCloud

springCloud是微服务架构，springCloud 提供了一站式的分布式微服务的解决方案。

springCloud和dubbo的区别？

共同点：
二者都是开源分布式服务的架构

不同点：

  1. 原理: dubbo属于RPC ，Cloud属于rest
  2. 背景: Cloud属于spring家族，具有强大的社区 
    dobbo是阿里巴巴后被停用，当当继续维护
    3.更新速度: 分别在社区中明显 Cloud的更新速度要快于dubbo，
    4.技术: Cloud提供了一站式的技术解决方案 
    dobbo 需要用到其他厂商提供的技术解决方案。

试题一

三张表，有主外键 分别为学生、班级、年级表。
求每个年级每个班级的总人数？

思路：求每个年级班级的人数。
     1.将年级、班级分组
     2.三表通过左外连接，设置条件
     3.查询年级、班级，和人数总和
     注： 1.GROUP BY和统计聚合函数同用时，
     除分组字段其他字段不可查询。
select 年级、班级 ,count(学号) 
from user a left join grade  b on a.`年级id`=b.`年级id`
left join class c on b.`班级id`=c.`班级id`
GROUP BY 年级、班级

数据库编程

发表于 2020-06-02 | 更新于 2020-06-07 | 分类于 数据库

数据库编程

在数据库的实际操作中，经常会有一个完整的操作需要多条语句来处理多个表才能为完成。例如，为了处理某个商品的订单，需要核对以保证库存中相应的商品，此时需要多条SQL语句来针对几个数据库表完成这个处理要求，而这个要求可以通过数据库编程来实现。

介绍常用的两种数据库编程技术，即存储过程与存储函数。

存储过程

在数据库系统中，随着功能不断丰富，系统变得越来越复杂，数据库开发人员会花费大量的时间和精力在SQL代码和应用程序的编写上。在多数情况下，许多代码会被重复使用多次，且每次都会输入相同的代码，这样既繁琐又会降低系统的运行效率。因此，需要提供一种方法，它可以将一些固定的操作集合起来，由数据库服务器来完成，实现某个特定任务，这就是存储过程。

什么是储存过程？

存储过程是一组为了完成某项特定功能的SQL语句集，其实质上就是一段经过编译后存储在数据库上的代码。用户只需要通过指定存储过程的名字并给定参数，即可以随时调用并执行它，而不需要重新编译，因此在这种通过定义一段程序存储在数据库中的方式，可以加大数据库操作语句的执行效率。

优点

1、可增强SQL语言的功能和灵活性
2、良好的封装性
3、高性能
4、可减少网络流量 
5、存储过程可作为一种安全机制来确保数据库的安全性和数据的完整性

创建

创建存储过程的几种方式：
1.命令行创建
  I.打开电脑的终端 输入 mysql -uroot -p 前提是安装了mysql    
  II.进入到该库中，delimiter// -->create procedure sp_name()-->begin-->select * from user;-->end-->//
  这样就创建了。（delimite命令用户可以定义结束符 ）
2.通过navicat工具创建
  a.在该库下查询中创建
    CREATE PROCEDURE sp_name(in cname varchar(200))

  b.在该库下的函数中创建
  使用工具时不需要创建存储过程配置下就行了，直接写储存过程体         begin
  select * from user 
  end

调用 (在程序或者其他存储过程中调用它）

CALL sp_name(parameter[...])

删除

DROPPROCEDURE [IF EXISTS]sp_name

存储函数

存储函数与存储过程一样，都是由SQL语句和过程式语句所组成的代码片段，并且可以被应用程序和其他SQL语句调用。

两者的区别

1.存储函数不能拥有输出参数，这是因为存储函数自身就是输出参数，而存储过程可以拥有输出参数
2.可以直接对存储函数进行调用，不需要使用CALL语句；而对存储过程可以拥有输出参数。
3.存储函数中必须包含一条RETURN语句，而这条特殊的SQL语句不允许包含存储过程中。

idea

idea对于程序员来说并不陌生，idea支持多种编程语言：Java、python、scnal等，
在开发中提供了便利，提高了开发的效率，作为当今社会的开发必备软件之一。

idea的下载及安装

下载

[idea官网](https://www.jetbrains.com/)中分为旗舰版和社区版，旗舰是收费的，社区版是免费的。旗舰版功能多于社区版的，结合自身下载所需，那么下载旗舰版的是可以使用30天的，若过期了怎么办呢？下面是idea 2019.3破解的亲测步骤。

安装

1.在下载的文件中找到bin,然后将下载好的破解文件放入的bin目录下
2.需要注意的是验证时，就选择免费使用30天，其他的就点继续

破解

资金充足的建议上官方网上购买
1.双击idea打开help-->edit custom vm options将最后加入
-javaagent:+破解文件的全路径（写错了话会打不开idea）保存
2. 重启idea 
3.打开help—-> register-->Activation code
  将网上找的激活码放上激活即可（有些网址不提供激活码了，要耐    心的着，破解文件和idea版本要符合）好啦！可以安心的敲代码啦。

遇到的问题

1.本机上下载过idea多个版本，卸载之后再下载即然打不开idea
  可能是idea 没有彻底的清除完和新的版本冲突就打不开。
  解决办法：
  打开终端输入：cd /Users/{你的电脑名}/ Library/ Preferences/，进入Preferences文件夹ls查看之前的版本 
   --删除 rm -rf IntelliJIdea2019.1  --重新安装idea
2.激活了后来又出现
Your activation code could not be validated(error 1653219)
“0.0.0.0 account.jetbrains.com”及“0.0.0.0  www.jetbrains.com”到hosts中屏蔽联网。
解决办法：
第一步将：0.0.0.0 https://account.jetbrains.com:443加入hosts，<br>不需要0.0.0.0  www.jetbrains.com，不然会影响idea的某些功能
